# Kim Dohoon 个人网站

这是一个适合 TWS 成员金道勋的追星博主使用的静态个人网站模板，适合记录：
- Instagram / Weverse 动态
- TMI 小细节
- 舞台与演出回忆
- 照片与文字记忆记录

## 直接查看

直接双击打开 index.html 即可查看页面。

## 可上传部署说明

这个项目已经整理为纯静态网页，可直接上传到以下平台：
- GitHub Pages
- Netlify
- Vercel

### 上传步骤（Netlify）
1. 打开 https://www.netlify.com/
2. 点击 Add new site -> Deploy manually
3. 拖入当前整个文件夹
4. 发布后即可获得公开链接

### 上传步骤（GitHub Pages）
1. 把当前文件夹内容上传到 GitHub 仓库
2. 打开仓库 Settings -> Pages
3. 选择 Deploy from a branch
4. 选择 main 分支 / 根目录
5. 生成公开地址

如果你想继续扩展，可以把内容替换成你自己的收藏和截图。
